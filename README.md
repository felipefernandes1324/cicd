Esta é uma API simples desenvolvida em Java usando Spring Boot. Ela possui um único endpoint que sempre retorna status HTTP 200 (OK). O objetivo principal é servir como base para a implementação e teste de pipelines CI/CD, garantindo que o serviço esteja sempre operacional e responda corretamente.

## Motivação

Ao criar uma API com um endpoint que sempre responde 200, você pode utilizar essa aplicação para:

- Testar pipelines de integração e entrega contínua (CI/CD) com facilidade.
- Garantir que a infraestrutura está funcionando corretamente antes de adicionar funcionalidades mais complexas.
- Facilitar a automação de deploys e monitoramentos básicos com um serviço estável e previsível.

## Tecnologias utilizadas

- Java 17
- Spring Boot 3.x
- Maven (como gerenciador de dependências)
